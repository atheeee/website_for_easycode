# website_for_easycode

Проект состоит из нескольких файлов: HTML‑страницы (index.html), таблицы стилей (styles.css), скрипта с кодом (interaction.js) и набора изображений. Для локального запуска можно использовать расширение Live Server в Visual Studio Code (в html файле нажат run with live server).
